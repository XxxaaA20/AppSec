*** Для корректной работы всех используемых настроек и компонентов нужно установить в Jenkins некоторые плагины, которые дадут нужный функционал и облегчат дальнейшую работу ***
*** Плагины можно будет установить при первом запуске Jenkins либо потом в менюшке Manage Jenkins -> Plugins ***

	******************************

- Build Timeout
This plugin allows you to automatically terminate a build if it's taking too long.

- Docker Pipeline
Build and use Docker containers from pipelines.

- Docker plugin
This plugin integrates Jenkins with Docker

- Generic Webhook Trigger
Can receive any HTTP request, extract any values from JSON or XML and trigger a job with those values available as variables. Works with GitHub, GitLab, Bitbucket, Jira and many more.

- Git Parameter
Adds ability to choose branches, tags or revisions from git repositories configured in project.

- GitLab
This plugin allows GitLab to trigger Jenkins builds and display their results in the GitLab UI.

- JavaMail API
This plugin provides the JavaMail API for other plugins.

- Job DSL
This plugin allows Jobs and Views to be defined via DSLs

- Locale
This plugin lets you control the language of Jenkins.

- OWASP Dependency-Track Plugin
This plug-in publishes Software Bill-of-Materials (SBOM) to Dependency-Track for continuous analysis. This plugin supports CycloneDX and SPDX SBOM formats.
Dependency-Track is an intelligent Software Supply Chain Component Analysis platform that allows organizations to identify and reduce risk from the use of third-party and open source components.

- Parameterized Trigger plugin
This plugin lets you trigger new builds when your build has completed, with various ways of specifying parameters for the new build.

- Pipeline
A suite of plugins that lets you orchestrate automation, simple or complex. See Pipeline as Code with Jenkins for more details.

- Pipeline: Stage View Plugin
Pipeline Stage View Plugin.

- SSH Agent
This plugin allows you to provide SSH credentials to builds via a ssh-agent in Jenkins.

- SSH server
Adds SSH server functionality to Jenkins, exposing CLI commands through it.

- Timestamper
Adds timestamps to the Console Output

- Workspace Cleanup Plugin
This plugin deletes the project workspace when invoked.
